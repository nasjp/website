Next.jsで必ずPrettierの設定をしているが、毎回調べるのが面倒なので、最低限の設定をまとめておく。

```bash
npm install --save-dev prettier-plugin-organize-imports
```

```bash
cat '{ "plugins": ["prettier-plugin-organize-imports"] }' > .prettierrc
```

```package.json
{
  "scripts": {
    "format": "prettier --write ."
  }
}
```
